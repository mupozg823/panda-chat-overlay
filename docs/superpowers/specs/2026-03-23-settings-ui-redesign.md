# 설정기 UI/UX 리디자인 스펙

> 작성일: 2026-03-23
> 상태: 승인됨

## 배경

full-injection 방식 전환으로 CSS 생성기에서 `compatMode`, `resetInner`, `themeReset` 로직이 제거됨. 이에 따라 설정기 UI도 불필요해진 호환성 섹션을 제거하고, 전체 구조를 개선한다.

## 결정 사항

| 항목           | 결정                          |
| -------------- | ----------------------------- |
| 전체 레이아웃  | 좌설정 + 우미리보기 유지      |
| 탭 구조        | 3탭: 프리셋 / 디자인 / 고급   |
| 디자인 탭 내부 | 접이식 섹션 3개               |
| 비주얼 스타일  | 미니멀 다크 (흑백 기반, 샤프) |
| CSS 복사       | 미리보기 상단 바에 통합       |

## 1. 비주얼 스타일 변경

### 컬러 시스템

현재 보라 그라데이션 기반(`--bg-main:#0f1120`, `--accent:#6c8cff`)에서 순수 흑백 기반으로 전환.

```
--bg-main: #000000
--bg-panel: #111111
--bg-surface: #1a1a1a
--border: #262626
--border-hover: #333333
--text-primary: #ffffff
--text-secondary: #a3a3a3
--text-muted: #525252
--text-dim: #333333
--accent: #ffffff (활성 탭, 선택 상태)
--accent-hover: #e5e5e5
```

### 타이포그래피

- 제목: `font-weight:700`, `letter-spacing:-0.5px`
- 본문/라벨: 현재 크기 유지
- 탭: `font-weight:600`, 활성 시 `border-bottom:2px solid #fff`

### 컴포넌트 스타일 변경

- 토글 스위치: 현재 보라(`#6c8cff`) 활성색 → 흰색(`#fff`)
- 버튼: 주요 CTA는 `background:#fff; color:#000`, 보조는 `border:1px solid #262626`
- 색상 입력: 현재 패턴 유지 (picker + hex), 배경을 `#1a1a1a`로
- range 슬라이더: thumb 색상 `#fff`, track `#262626`
- 섹션 접기: 현재 `▾` 패턴 유지, 색상만 조정
- 프리셋 카드: 컬러 스와치 세로바 → 상단 가로바, `border:1px solid #262626`, 선택 시 `border-color:#fff`

## 2. 탭 구조 재편

### 현재 → 변경

```
현재:
  프리셋 | 레이아웃 | 세부 조정 (7섹션)

변경:
  프리셋 | 디자인 | 고급
```

### 프리셋 탭 (변경 최소)

- 카테고리 필터: 전체/기본/감성/다크 (현재 유지)
- 프리셋 그리드: 2열 (현재 3열 → 2열로 변경, 각 카드에 정보 더 표시)
- 프리셋 설명 박스 유지

### 디자인 탭 (레이아웃 + 세부조정 일부 통합)

접이식 섹션 3개:

**말풍선** (초기 펼쳐짐)

- 배경 색상 + 투명도
- 모서리 둥글기
- 배경 흐림
- 테두리 (두께 + 색상)
- 그림자 (크기 + 색상)
- 그라데이션 (토글 + 시작/끝 색상 + 방향)

**레이아웃** (초기 접혀짐)

- 메시지 스타일 (전체말풍선/닉캡슐형/닉본문분리형) — 현재 레이아웃 탭의 카드 UI를 여기로 이동
- 줄바꿈 토글
- 메시지 간격
- 안쪽 여백 (paddingX)
- 최대 너비
- 구분자 텍스트
- 텍스트 배경 (색상, 투명도, 둥글기, 패딩)

**아이콘 & 캡슐** (초기 접혀짐)

- 닉네임 아이콘 선택
- 아이콘 위치 (기본/왼쪽아바타)
- 커스텀 이미지 (URL/파일)
- 아이콘 크기
- 아바타 프레임 (형태, 배경, 테두리, 그림자)
- 캡슐 배경 (색상, 투명도, 그라데이션)
- 캡슐 둥글기

### 고급 탭 (세부조정 나머지)

접이식 섹션 4개:

**후원 & 알림** (초기 펼쳐짐)

- 후원: 배경색, 투명도, 이미지 크기, 금액 강조색, 텍스트 색상, 둥글기, 이미지 숨기기, 전체 숨기기
- 알림: 배경색, 투명도, 텍스트 색상, 둥글기, 전체 숨기기

**등급별 배지** (초기 접혀짐)

- 사용 토글
- 배지 크기
- 등급별 커스텀 그리드

**표시 설정** (초기 접혀짐)

- 대상 테마 선택 (compatTheme — 유지, 어떤 테마에 CSS를 적용할지 선택)
- 상단 바 숨기기
- 아이콘 숨기기
- 페이드아웃 방지
- 애니메이션 제거

**추가 CSS** (초기 접혀짐)

- 커스텀 CSS 텍스트에어리어 — 현재 CSS 출력 영역에 있던 것을 고급 탭으로 이동

## 3. 미리보기 영역 재구성

### 상단 바

```
[미리보기]  [테마선택▾]  [배경●●●]  |  [CSS 복사]  [설정 ▾]
```

- 왼쪽: "미리보기" 라벨 + 테마 선택 드롭다운 + 배경 선택 도트
- 오른쪽: CSS 복사 버튼 (주요 CTA, `background:#fff`) + 설정 관리 드롭다운
- 현재 `preview-meta` (CSS 대상/호환 모드 텍스트) → 제거 (호환 모드 개념 제거됨)
- 현재 `preview-quick-dock` → 유지 (말풍선/닉네임/본문/아바타 퀵 편집)

### CSS 출력 영역

- 미리보기 하단에 접기/펼치기 가능한 코드 영역
- 기본 접혀짐, 한 줄 미리보기 표시 (`/* PandaTV Chat Overlay - full-injection output */`)
- 펼치면 현재처럼 코드 표시 (max-height 제한)
- OBS 가이드는 제거하거나 도움말에 통합

## 4. 제거 목록

| 제거 대상                                              | 이유                                           |
| ------------------------------------------------------ | ---------------------------------------------- |
| 호환성 섹션 (`#sec-compat`)                            | full-injection으로 불필요                      |
| `compatMode` 셀렉트                                    | CSS 생성기에서 제거됨                          |
| `resetInner` 토글                                      | CSS 생성기에서 제거됨                          |
| `themeReset` 토글                                      | CSS 생성기에서 제거됨                          |
| 퀵 네비 바 (`quick-nav`)                               | 섹션 7→3개로 축소, 불필요                      |
| 레거시 대시보드 숨김 요소들                            | `phaseOverviewLabel`, `phasePrevBtn` 등 미사용 |
| `preview-meta` 텍스트                                  | 호환 모드 표시 제거                            |
| `previewChat.dataset.compatMode/resetInner/themeReset` | 미사용                                         |
| OBS 가이드 (`obs-guide`)                               | 도움말로 통합 또는 제거                        |
| 보라 그라데이션 CSS 변수                               | 새 컬러 시스템으로 대체                        |

## 5. 유지 목록

| 유지 대상            | 비고                                                   |
| -------------------- | ------------------------------------------------------ |
| `compatTheme` 셀렉트 | 대상 테마 선택 기능은 유지 (고급 > 표시 설정으로 이동) |
| 플로팅 에디터        | 미리보기 클릭 편집 기능 유지                           |
| 드래그 핸들          | 미리보기 인터랙티브 편집 유지                          |
| 설정 저장/불러오기   | localStorage + JSON 파일 입출력 유지                   |
| 반응형 레이아웃      | 1200px/720px 브레이크포인트 유지                       |

## 6. 파일 영향 범위

| 파일                       | 변경 범위                                                        |
| -------------------------- | ---------------------------------------------------------------- |
| `overlay-settings.html`    | HTML 구조 재편 (탭, 섹션, 상단바), 인라인 JS 동기화              |
| `styles/settings.css`      | 컬러 시스템 전면 교체, 컴포넌트 스타일 업데이트                  |
| `scripts/preview.js`       | `compatMode`/`shellPreview` 참조 정리 (대부분 완료), 상단바 연동 |
| `scripts/state.js`         | `compatMode`/`resetInner`/`themeReset` 기본값 정리               |
| `scripts/ui-utils.js`      | 탭 전환 로직 변경 (layout→design, detail→advanced)               |
| `scripts/css-generator.js` | 이미 완료                                                        |
| `scripts/preset-ui.js`     | 프리셋 그리드 레이아웃 변경 (3열→2열)                            |

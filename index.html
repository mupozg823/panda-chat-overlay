PandaTV Chat Overlay

v0.2.0

팬더TV 자체에서 지원하지 않는 채팅창 커스터마이징 기능을 제공합니다.
말풍선, 닉네임 캡슐, 그라데이션 배경 등 플랫폼 기본 설정으로는
불가능한 세밀한 채팅창 꾸미기를 설정기에서 조정하고, OBS에 붙여넣으세요.

설정기 열기 레퍼런스 프리셋으로 열기

사용법

  1. Windows: start.bat 더블클릭
  2. Mac: start.command 더블클릭
  3. 또는 overlay-settings.html을 브라우저에서 직접 열기
  4. 설정 완료 후 CSS 복사하기 클릭
  5. OBS → 브라우저 소스 → 사용자 지정 CSS에 붙여넣기

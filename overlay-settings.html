용어 설명

안정형 / 균형형 / 원형 유지

안정형은 원본 효과를 정리해서 가장 무난. 균형형은 원본 느낌 일부 유지. 원형 유지는 원본 구조 최대한 보존.

전체 말풍선 / 캡슐형 / 분리형

전체 말풍선이 가장 안정적. 캡슐형은 닉네임 강조. 분리형은 디자인 효과가 크지만 설정이 더 필요.

팬더TV 채팅 오버레이

OBS 채팅 오버레이 커스텀 CSS 생성기

이 도구는 OBS 브라우저 소스에 붙여넣을 커스텀 CSS를 만들어 줍니다.
팬더TV 채팅 위젯의 기본 스타일을 덮어쓰므로, 스타일 조정은 이 설정기에서만 하세요.

처음이세요? 3단계로 끝!

1
프리셋 고르기

왼쪽 프리셋 탭에서 마음에 드는 스타일을 클릭하면 즉시 미리보기에 반영됩니다.

2
미리보기에서 직접 편집

오른쪽 미리보기 영역의 채팅 메시지를 클릭하면 해당 부분의 세부 설정을 바로 조정할 수 있습니다. 색상, 크기, 둥글기 등을 실시간으로 확인하세요.

3
CSS 복사 → OBS 붙여넣기

상단의 CSS 복사 버튼을 누르고, OBS 브라우저 소스의 "사용자 지정 CSS" 칸에 붙여넣으면 완료!

OBS 브라우저 소스 설정법

A
브라우저 소스 추가

OBS에서 소스 추가 → 브라우저 선택 → 이름 입력 후 확인

B
URL 입력

팬더TV 홈페이지 → 마이페이지 → 채팅 위젯 → 위젯 URL 복사 → OBS URL 칸에 붙여넣기

C
크기 설정

미리보기 아래 OBS 브라우저 소스 크기 가이드를 참고하세요 — 설정에 맞는 최적 크기가 자동 계산됩니다.

D
커스텀 CSS 붙여넣기

"사용자 지정 CSS" 칸의 기존 내용을 모두 지우고, 이 설정기에서 복사한 CSS를 붙여넣으세요.

설정 패널 구성

프리셋 — 시작점 선택

32종의 완성된 스타일 중 하나를 골라 시작하세요. 기본은 범용, 귀여운 & 감성은 여성 스트리머향, 다크 & 특수는 게임/감성 방송용입니다.

디자인 — 세부 조정

말풍선 색상·투명도·둥글기, 텍스트 크기·색상, 후원/알림 스타일, 아바타 아이콘 등을 세밀하게 조정합니다. 상단의 말풍선 / 닉네임 / 본문 버튼으로 빠르게 이동할 수 있습니다.

고급 — 전문 설정

호환 모드, 테마 대상, 패딩, 줄 높이 등 고급 옵션입니다. 보통은 건드릴 필요 없습니다.

미리보기 활용법

메시지 클릭 → 플로팅 편집기

미리보기의 채팅 메시지, 후원, 알림을 직접 클릭하면 해당 영역의 설정을 슬라이더로 바로 조정할 수 있습니다. 가장 직관적인 편집 방법입니다.

테마 미리보기 전환

상단의 미리보기 드롭다운에서 팬더TV 테마(default, kakaotalk, neon 등)를 바꿔 각 테마에서 어떻게 보이는지 확인할 수 있습니다.

후원 · 알림 설정

후원 메시지 꾸미기

디자인 탭 → 후원 섹션에서 배경색, 투명도, 둥글기, 텍스트 색상, 이미지 크기를 조정합니다. 후원 이미지 숨기기를 켜면 이미지 없이 텍스트만 표시됩니다.

알림 메시지 꾸미기

추천, 입장 등의 알림 메시지 스타일을 조정합니다. 배경색·글자색·둥글기를 설정하고, 불필요하면 알림 전체 숨기기로 끌 수 있습니다.

용어 설명

안정형 / 균형형 / 원형 유지

안정형: 원본 효과를 깔끔하게 정리 (추천). 균형형: 원본 느낌 일부 유지. 원형 유지: 원본 구조 최대한 보존 (고급).

전체 말풍선 / 캡슐형 / 카드형

전체 말풍선: 닉+메시지를 하나의 버블로 감쌈 (가장 안정적). 캡슐형: 닉네임만 별도 캡슐로 강조. 카드형: 실제 위젯 흐름은 유지하고 본문만 카드처럼 강조.

호환 테마

전체: 모든 팬더TV 테마에 적용 (추천). 특정 테마만 선택하면 해당 테마에서만 CSS가 적용됩니다.

페이드 방지 / 애니메이션 제거

페이드 방지: 채팅이 시간이 지나면 사라지는 효과를 끕니다. 애니메이션 제거: 메시지 등장 시 슬라이드/페이드 효과를 끕니다. 둘 다 OBS에서 깔끔한 표시를 위해 켜는 것을 추천합니다.

자주 묻는 질문

OBS에서 적용했는데 안 바뀌어요

1) OBS 브라우저 소스의 URL 칸에 복사한 URL을 붙여넣었는지 확인하세요. 2) 브라우저 소스를 우클릭 → 캐시 새로고침을 눌러보세요. 3) 팬더HP 채팅 설정에서 "위젯 테마"가 이 도구의 "호환 테마" 설정과 일치하는지 확인하세요.

글자색이 미리보기와 다르게 나와요

팬더HP 채팅 설정의 글자색·폰트가 CSS보다 먼저 적용됩니다. 팬더HP에서 기본 설정을 맞추고, 이 도구에서 오버레이 효과를 덮어쓰세요.

설정을 저장/불러오기 하고 싶어요

미리보기 옆의 설정 관리 버튼에서 현재 설정을 JSON 파일로 내보내거나, 이전에 저장한 파일을 불러올 수 있습니다.

초기화하고 싶어요

상단의 초기화 버튼을 누르면 모든 설정이 기본값으로 돌아갑니다. 실수로 눌렀다면 Ctrl+Z(되돌리기)로 복구할 수 있습니다.
